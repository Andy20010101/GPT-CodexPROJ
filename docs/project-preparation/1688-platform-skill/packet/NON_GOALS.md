# Non Goals

This file exists to keep the first run from expanding into adjacent work.

## Explicit Non Goals

- 不是要在第一版里做一个全站 1688 爬虫框架。
- 不是要在第一版里解决验证码、代理池、封禁恢复、账号体系或高并发采集问题。
- 不是要在第一版里建立供应商历史数据库、价格时序库或长期监控系统。
- 不是要在第一版里做自动联系工厂、自动询盘、自动下单或自动谈判。
- 不是要在第一版里改造 OpenClaw 宿主平台本身、增加新的插件系统或引入新的运行时。

## Defer Until Later

- 把 Markdown 输出再扩展成 CSV、JSON、表格同步或数据库落盘。
- 引入更复杂的质量评分模型、可配置打分权重或跨会话供应商记忆。
- 支持多个 marketplace 或多个供应链场景。
- 研究是否需要官方 API、浏览器扩展、独立服务或更重的抓取基础设施。

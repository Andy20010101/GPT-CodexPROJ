# v0.1.0

## Highlights

- First public baseline release of `review-then-codex-system`
- Three-plane monorepo with orchestrator control plane, ChatGPT web bridge review plane, and replaceable execution plane
- Preparation-layer workflow, bounded self-improvement tooling, and local operator documentation are now part of the published repo surface

## Included Areas

- Repository surface:
  - root skill entrypoint in `SKILL.md`
  - contributor, security, changelog, and release-process docs
  - `.env.example`, issue templates, and PR template
- Preparation and planning:
  - project preparation workflow spec, SOP, CLI docs, templates, examples, and sample preparation packets
  - downstream planning examples under `docs/downstream-planning/`
- Runtime:
  - hardened orchestrator workflow runtime, review evidence rules, recovery, daemon controls, and workspace lifecycle handling
  - hardened bridge browser/session control, diagnostics, readiness probing, and conversation control surfaces
- Repository boundaries:
  - nested project repos removed from the monorepo
  - legacy imported material archived under `references/legacy/`

## Validation

Release baseline validated with:

```bash
npm run lint
npm run typecheck
npm test
```

## Known Limits

- Local-first operating model only
- Real bridge usage still assumes an already logged-in ChatGPT browser session
- The orchestrator is not a distributed scheduler or HA workflow engine
- Root and workspace packages remain `private: true`; this release does not publish npm artifacts

## Upgrade Notes

- Use GitHub Releases and annotated tags as the authoritative release surface
- Treat `v0.1.0` as the first public baseline rather than a retroactive tag for older internal history
